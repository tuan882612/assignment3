## How to run

1. Create a virtual environment

    ```bash
    python -m venv venv
    ```

2. Activate the virtual environment

    ```bash
    source venv/bin/activate
    ```

3. Run the application

    ```bash
    # optional to remove python cache files
    make clean

    make run
    # or 
    python src/main.py
    ```
